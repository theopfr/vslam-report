# Interactive Guide to vSLAM (_visual Simultaneous Location And Mapping_)

This [blog](theopfr.github.io/vslam-report/) walks through how a single video becomes a camera trajectory and a 3D map of its surroundings, combining interactive demos with technical explanations.

### Steps covered:
1. ORB feature detection
2. Feature matching
3. Camera motion estimation
4. Triangulation
5. 3D point cloud map
6. Loop closure (bonus)

---
